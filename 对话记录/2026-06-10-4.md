找到了问题所在。ant-design-vue 的 `.ant-modal-footer` 使用固定颜色 `#e8e8e8` 作为顶部分割线，在暗色模式下没有适配。需要在全局样式中添加暗色模式下的分割线颜色覆盖。



已修复暗色模式下上传作品弹窗底部分割线颜色未适配的问题。

**问题原因**：ant-design-vue 的 `.ant-modal-footer` 默认使用固定颜色 `#e8e8e8` 作为顶部分割线，在暗色模式下未自动适配。

**修复方案**：在 `global.css` 中添加了暗色模式下的样式覆盖：

```css
[data-theme='dark'] .ant-modal-footer {
  border-top-color: var(--border-secondary);
}
```

这样分割线颜色会使用 CSS 变量 `var(--border-secondary)`，在暗色模式下该变量值为 `#2C2C2C`，与深色背景更加协调一致。
